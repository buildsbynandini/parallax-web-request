User types URL

Step 1: Enter URL

You type a URL in the browser, for example: www.example.com

DNS Lookup

Step 2: DNS Lookup

The browser translates the domain name to an IP address through the DNS system.

Server Request

Step 3: Server Request

The browser sends a request to the server at that IP address asking for the webpage.

Server Response

Step 4: Server Response

The server processes the request, fetches data, and sends back the webpage content.

Page Rendered

Step 5: Page Rendered

The browser renders the HTML, CSS, and JS, displaying the webpage on your screen.
